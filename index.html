⚡ Step 1 — Register Azure AD App (2 min)

  1. Go to portal.azure.com → Azure Active Directory → App registrations
  2. Click + New registration → Name: AVPMi Dashboard
  3. Account type: Single tenant
  4. Platform: Single-page application (SPA)
    Redirect URI: http://localhost:8080
  5. Click Register → Copy the Application (client) ID
  6. Under API permissions → Add → SharePoint → Delegated → AllSites.Read
  7. Click Grant admin consent

🔧 Step 2 — Configure this dashboard (1 min)

  1. Open this HTML file in a text editor
  2. Find the line: clientId: "",
  3. Paste your Application (client) ID between the quotes
  4. Save the file
  5. Run from terminal: python -m http.server 8080 in the same folder
  6. Open: http://localhost:8080/AVPMi_Closed_Projects_Dashboard.html
  7. Click Connect Live Data in the header

💡 For CEO/CFO access: Upload this file to your SharePoint document library after setup. Set the Redirect URI to that SharePoint URL. They'll auto-authenticate with their M365 accounts.
